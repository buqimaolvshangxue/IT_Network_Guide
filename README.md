# IT 网络知识完备指南

> 从零开始，让你成为一个胸有成竹的 IT。

## 这是什么

一份面向网络初学者的完整教程。读完之后你能：

- 理解网络中每个设备、每条线、每个配置项的含义
- 独立完成路由器配置、VPN 搭建、代理搭建、内网穿透、域名管理、服务部署
- 网络出问题时，有系统的排查思路
- 听别人讨论网络话题时，不会出现"完全听不懂的词"

## 怎么读

**如果你是零基础**：从第 0 章开始，按顺序读。每章只依赖前面的知识。

**如果你有一些基础**：看目录，跳到你不懂的章节。

**如果你在解决具体问题**：翻附录 B（常见故障速查）或附录 A（命令速查表）。

## 新手最短路径（推荐）

如果你是第一次系统学网络，建议按这条最短路径走：

1. [第 0 章](00_overview.md)（先建立全局图）
2. [第 1 章](01_devices_and_identity.md) + [第 2 章](02_automation_and_discovery.md)（先搞懂 IP/DHCP/DNS）
3. [第 3 章](03_transport_and_protocols.md)（端口 + TCP/UDP）
4. [第 7 章](07_home_network.md)（先把家里网络配到可用）
5. [第 8 章](08_remote_access.md)（先跑通 SSH）
6. [第 10 章](10_troubleshooting.md)（建立排障习惯）

> 💡 每章开头都有 `🚦` 快速入口（30 秒先懂 / 最小可用 / 阅读顺序），先看这个再读正文。
