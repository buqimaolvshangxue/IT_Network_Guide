## 学习路径建议

### 入门阶段（1-2 周）
1. 观看"计算机网络微课堂"前 10 集，理解基础概念
2. 学习 Linux 命令行基础（文件操作、权限、进程）
3. 配置 SSH 密钥登录，熟悉 SSH 基本用法

### 进阶阶段（2-4 周）
1. 搭建 WireGuard VPN，实现设备互联
2. 使用 Caddy 配置反向代理和 HTTPS
3. 学习 Docker 基础，运行常见服务（Nginx、MySQL）

### 实战阶段（持续）
1. 搭建家庭实验室（NAS、软路由、内网穿透）
2. 遇到问题时使用排查工具（ping、traceroute、tcpdump）
3. 参与社区讨论（恩山论坛、V2EX、Reddit）

## 持续学习建议

1. **订阅技术博客**：阮一峰的网络日志、少数派、云原生社区
2. **关注 GitHub 项目**：Star 感兴趣的开源项目，阅读 README 和 Wiki
3. **动手实践**：理论结合实践，每学一个概念就搭建一个小项目
4. **记录笔记**：使用 Markdown 记录配置文件和排查过程，方便日后查阅
5. **参与社区**：在论坛回答问题，教学相长，加深理解
